
Чтобы импортировать в Базу материалов и комплектующих материалы из файла hettich.dbf, следует выполнить следующие действия.

1. Вызовите команду Настройка - База материалов - Импорт номенклатуры из DBF.
2. В появившемся на экране диалоге База материалов и комплектующих вызовите команду Файл - Импорт номенклатуры из DBF.
3. В диалоге Импорт номенклатуры из DBF нажмите кнопку Открыть и укажите файл hettich.dbf, который сохранен в папке /Hettich/База материалов/. 
4. Назначьте соответствие полей следующим образом:
 ITEMCODE - Артикул материала
 ITEMNAME - Наименование материала
 GRP_NAME - Наименование группы
 UNITNAME - Единица измерения
5. Нажмите кнопку ОК, чтобы завершить настройки импорта.
6. В диалоге Выберите группу щелкните по имени группы, в которой будет создана группа импортируемых матеиалов и нажмите кнопку ОК.
 Прогресс-индикатор в диалоге Импорт номенклатуры из DBF будет показывать степень выполнения операции. Она может занять некоторое время.
 После завершения операции сведения о количестве импортированных материалов будут показаны в диалоге Информация.
 Группа Hettich появится в выбранной группе базы данных.
 Подробно импорт данных в Базу материалов и комплектующих рассматривается в документе База данных материалов системы БАЗИС Руководство пользователя.

Чтобы обеспечить доступность библиотек из архива Hettich в системе БАЗИС, сохраните файлы из распакованного архива по следующим папкам:
  Файл  hettich.bdl из папки \Hettich\Библиотека дверей\ в папку, назначенную для библиотек дверей (раздел Библиотеки дверей (*.bdl) диалога Настройки).
  Файл  hettich.bbb из папки \Hettich\Библиотека ящиков\ в папку, назначенную для библиотек ящиков (раздел Библиотека ящиков (*.bbb, *.bb) диалога Настройки).
  Файлы  из папки \Hettich\Фрагменты\ в папку, назначенную для фрагментов (раздел Фрагменты (*.fr3d, *.frw) диалога Настройки).
  Файлы из папки \Hettich\Фурнитура\ в папку, назначенную для крепежных и фурнитурных изделий (раздел Крепеж, фурнитура (*.f3d) диалога Настройки).
